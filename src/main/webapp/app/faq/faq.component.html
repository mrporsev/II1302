FAQ!

    SETUP & INSTALLATION
  1. How to get started with Plantalyzer, what I do when I have just become a user?
    all you need to do after creating an account with us is to register your sensor on your account and put the sensor into the plant pot. The sensor will start sending data to your Plantalyzer account. Log in to your account to see the data.
  2. how do I get a sensor?
    we have not built a real sensor for Plantalyzer because we have no TIEDB students in the group, we will instead fake data for the sensor.
  3. how much does it cost (the sensor and the application)?
    Because we do not have a real sensor so we have no idea what the sensor cost but the application does not cost anything, it is free.
  4. What do I need to create a Plantalyzer account?
    All you need is a valid e-mail address to create an account in Plantalyzer
  5. how do I set up an account?
    The steps are shown below the question list
  6. how do I register the sensor to my account?
    Click on entities in the navbar --> sensor ---> add new sensor
  7. Is there any customer support for Plantalyzer?
    unfortunately, we do not provide this service
    USE & MAINTENANCE
  1. can I change the name of a sensor?
    Yes, go to the sensor page, click on the sensor ---> edit
  2. what are the recommended values?
    The answer to this question is board because the recommended values depend on the kind of this plant, for more info about sensor values, different plant categories, and their recommended values please visit the Plant care page
    INTEGRITY AND DATA PRIVACY
  1. How will user information be stored?
    The user information will be saved in our database.
  2. what user information is stored and for how long?
    Username, password, and information about the user's plant/s. While Plantalyzer is still in its prototype phase we have not implemented automatic deletion of older data
  3. will user information be sold or given to third parties?
    Your personal data will not be sold or shared with third parties
  4. if I want to keep my data under my own control, can I run the sensor on my own server?
    unfortunately not, you have to use both the sensor and the Plantalyzer web app.
