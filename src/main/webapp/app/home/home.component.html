Plantalyzer is a sensor, an IOT based application, which stores and presents data from a plant's environment. It's free to create an account but to be able to use the website's functions to their fullest extent you need to have a sensor installed beforehand. To view more information about how to install a sensor and what Plantalyzer is, go to the navigation bar and click on FAQ(Frequently Asked Questions) or About us.

This product has been developed by a group of students at KTH as an assignment in a course. Please note this webpage has been made for educational purposes only and that Plantalyzer won't be available to the general public as a product.
